데이터 타입

데이터 타입이란 값으 종류를 의미한다.
7개의 데이터 타입이 있고, 원시 타입과 객체 타입으로 분류할 수 있다.

원시타입
1. 숫자 타입(number type) 	: 숫자, 정수와 실수 구분 없이 하나의 숫자 타입만 존재
2. 문자열 타입 (string type)	: 문자열
3. 불리언 타입 (boolean type)	: 논리적 true & flase 
4. undefind 타입		: var 키워드로 선언된 변수에 암묵적으로 할당되는 값
5. null 타입			: 값이 없다는 것을 의도적으로 명시할 때 사용하는 값
6. 심벌 타입 (symbol type)	: ES6에서 추가된 7번째 타입

객체타입
- 객체, 함수, 배열 등


6.1 숫자타입

숫자 타입은 하나의 타입만 존재하며, 모든 수를 실수로 처리한다.
ex) console.log(1 === 1.0) -> true

숫자 타입은 세 가지의 특별한 값도 존재한다.
Infinity : 양의 무한대
-Infinity : 음의 무한대
NaN : 산술 연산 불가능 (not - a - number)

ex)
console.log(10/0); -> Infinity
console.log(10/-0); -> -Infinity
console.log(1*'String') -> NaN

*NaN의 경우 js에서는 대소문자를 구별하기 때문에 NaN를 예약어로 인식한다.



6.2 문자열 타입

문자열 타입은 텍스트 데이터를 나타내는 데 사용한다.
문자열은 '', "", `` 으로 감싼다.

문자열로 감싸지 않으면 식별자로 인식한다.



6.3 템플릿 리터럴

템플릿 리터럴 : ES6 부터 새로운 문자열 표기법이 도입되었는데 이는 멀티라인 문자열, 표현식 삽입,태그드 템플릿 등 편리한 문자열 처리 기능을 제공한다. 
템플릿 리터럴은 런타임에 일반 문자열로 변환되어 처리된다.

*멀티라인 문자열이란 ?
백틱 내부에서의 줄바꿈이 그대로 런타임에 인식된다.

*태그드 템플릿이란 ?

(예제코드)
let var1 ='var1';
let var2 = 2222;
let var3 = 3333;
let instans = transStr`첫번째 문자열 
${var1} 두번째 문자열 ${var2} 세번째 문자열 ${var3}`;
console.log(instans);

function transStr(str, ...rest){
    console.log(str);
    console.log(rest);
    return null;
}

(결과)
[ '첫번째 문자열 \n', ' 두번째 문자열 ', ' 세번째 문자열 ', '' ]
[ 'var1', 2222, 3333 ]
null

문자열은 str에 배열로 표현식은 rest에 배열로 나타낸다.
그리고 멀티라인인 경우에는 인덱스 값에 \n로 표현된다.
또한 문자열의 마지막 인덱스는 항상 공백이다.



6.4 불리언 타입

불리언 타입의 값은 논리적 참, 거짓을 나타내는 true와 false 뿐이다.



6.5 undefined 타입

var 키워드로 선언한 변수는 암묵적으로 undefined로 초기화 된다.
즉, 변수를 참조했을 때 undefined가 반환된다면 변수가 선언 이후 값이 할당된 적이 없는,
초기화 된 적이 없는 변수라는 것을 알 수 있다.


*자바스크립트는 대표적인 동적 타입 언어이다.
*동적 타입 언어란 ?
자바스크립트의 변수는 선언이 아닌 할당에 의해 타입이 결정된다. (타입추론)
그리고 재할당에 의해 변수의 타입은 언제든지 동적으로 변할 수 있다.
*정적 타입 언어와 동적 타입 언어
정적타입 -> java, C, C++ 등등
동적타입 -> java script, python, PHP 등

동적 타입 언어는 변수에 어떤 데이터 타입의 값이라도 자유롭게 할당할 수 있다. 이는 곧 데이터 타입에 대해 무감각해질 정도로 편리하다. 하지만
변수의 값은 언제든지 변경될 수 있기 때문에 복잡한 프로그램에서는 변화하는 변수의 값을 추적하기 어려울 수 있고, 변수의 값을 확인하기 전에는 타입을 확신할 수 없다. (타입이 언제든지 변할 수 있기 때문)
