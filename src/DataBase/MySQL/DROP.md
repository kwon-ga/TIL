## ✔ DROP

<br>

DROP 문을 사용해서 데이터베이스와 테이블을 삭제할 수 있다.

<br>

### ✔ 데이터베이스 삭제  


```sql
DROP DATABASE 데이터베이스이름
```

데이터베이스 자체를 삭제한다.

<br>

### ✔ 테이블 삭제  


```sql
DROP TABLE 테이블이름
```

데이터베이스 내부의 테이블 자체를 삭제한다.

<br>

### ✔ 테이블의 데이터만 삭제 


```sql
TRUNCATE TABLE 테이블이름
```
테이블 자체는 남게되며, 해당 테이블에 저장된 데이터만 모두 삭제한다.
